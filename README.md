# MULTIMEDIJSKA ZGODBA - NAČRT

## Tema in novičarska vrednost

Izbrali smo temo “Mladi in stanovanjska stiska v Sloveniji”. Mladi v Sloveniji se soočajo z enim najvišjih indeksov stanovanjske nedostopnosti v EU — najemnine rastejo, plače pa ne sledijo. Zgodba je pomembna, ker zadeva praktično vsako mlado osebo v državi.

## Ciljna publika

### Kdo

Mladi med 18–35 let, študenti, iskalci prve zaposlitve, mladi pari.

### Kje

Predvsem digitalno — Instagram, TikTok, spletni portali.

### Navade

Kratke vsebine, vizualno, osebne zgodbe jim pomenijo več kot statistika.

## Zaporedje branja

### Pričetek s spremljevalnim prispevkom

Intervjuji z mladimi, ki iščejo stanovanja in seznanjanje s problematiko iz prve roke.

### Glavni prispevek

Obrazložitev problematike in zapis članka.

### Infografika

Vizualna predstavitev podatkov in problematike.

### Interaktivni kalkulator dostopnosti stanovanja

Izračun koliko plače ostane posamezniku po plačilu mesečnega obroka stanovanja.

## Načrt formatov

Prispevek bomo naredili v obliki spletne strani oziroma bloga. Za izdelavo bomo uporabili spletne tehnologije HTML, CSS in JavaScript. Infografike bomo izrisovali s pomočjo knjižnic Chart.js oziroma D3.js.

## Izvirni element

Naš izvirni element bo sprogramiran spletni kalkulator. Uporabnik bo vnesel vrednost željene nepremičnine, svojo neto plačo in število obrokov. Kalkulator bo nato izračunal, koliko od plače ostane, ter ovrednotil, ali je to dovolj za preživetje posameznika.

Dodali bi lahko tudi druge možnosti, ki povečajo zanimivost kalkulatorja, na primer izris kvadrature hiše, ki jo uporabnik lahko postavi s svojo plačo, ali druge podobne vizualizacije.

## Načrt UI porabe

Uporabljali bomo razna UI orodja, predvsem Claude.ai, Gemini in ChatGPT, glede na preference posameznega člana skupine. Z njimi si bomo pomagali pri pisanju vsebine, postavitvi spletne strani, oblikovanju in programiranju.
